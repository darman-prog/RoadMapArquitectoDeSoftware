🧩 Modelo de datos
export interface Empleado {
  idEmpleado: number;
  nombre: string;
  departamento: string;
  sueldo: number;
}

📡 Servicio EmpleadoServicio

Crea un servicio con HttpClient usando inject() y una base URL fija http://localhost:8080/api/empleados.

Los métodos deben ser:

obtenerEmpleados(): Observable<Empleado[]>

obtenerEmpleadoPorId(id: number): Observable<Empleado>

agregarEmpleado(empleado: Empleado): Observable<Empleado>   // quita idEmpleado antes de enviar

editarEmpleado(id: number, empleado: Empleado): Observable<Empleado>  // manda objeto completo

eliminarEmpleado(id: number): Observable<{message: string}>


👉 Regla: usar inject() en lugar de constructores donde sea posible.
